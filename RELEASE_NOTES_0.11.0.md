# 箱書きエディタ v0.11.0

**リリース日**: 2025年12月18日

## ✨ 主な新機能

### 📝 執筆エディタの実装
箱書きから直接執筆画面へ遷移できるようになりました。

- **シームレスな執筆体験**: シーンカードの「執筆」ボタンからエディタへ遷移
- **文字数カウント**:
    - **進捗管理**: 「本文文字数」と、その日の「進捗（+◯◯文字）」を表示
    - **スマートカウント**: 箱書き部分のセパレーター以降の本文のみをカウント
- **カスタマイズ**:
    - フォント（ゴシック、明朝、等幅）の切り替え
    - フォントサイズの変更
- **安心の保存機能**:
    - **自動保存**: 入力停止から2秒後に自動的に上書き保存
    - **データ永続化**: ページ遷移やリロードでも入力内容を保持
    - **離脱確認**: 未保存の変更がある場合、一覧に戻る際に警告を表示

### 📂 インテリジェントなファイル追跡システム
書き出されたファイルの管理が大幅に改善されました。

- **絶対的なシーン番号**: 各シーンに一意の番号を付与
- **章ごとの固定フォルダ番号**: 章の移動があってもフォルダ構成を維持
- **自動同期**: 
    - 章の移動やタイトル変更に合わせてファイル名・パスを自動更新
    - 執筆中の本文を完全に保護しながら自動移動

### ⚙️ 設定画面の刷新
設定項目が増えたことに伴い、設定画面を整理しました。

- **タブ表示**: 「基本設定」「箱書き」「エディター」にタブを分類
- **基本設定**: テーマ（ライト/ダーク/システム）、自動保存のON/OFF
- **箱書き**: 時間・場所の入力モード設定
- **エディター**: フォント・フォントサイズ設定

## インストール方法

### macOS
1. DMGファイルをダウンロード
2. DMGファイルを開く
3. アプリケーションフォルダにドラッグ&ドロップ

**注意**: 初回起動時に「開発元が未確認」の警告が出る場合：
1. アプリケーションを右クリック（またはControlキー+クリック）
2. 「開く」を選択
3. 警告ダイアログで「開く」をクリック

### Windows
1. インストーラー（.exe または .msi）をダウンロード
2. ダウンロードしたファイルを実行
3. インストールウィザードの指示に従う

### Linux (Debian/Ubuntu)
```bash
sudo dpkg -i 箱書きエディタ_0.11.0_amd64.deb
```

### Linux (AppImage)
```bash
chmod +x 箱書きエディタ_0.11.0_amd64.AppImage
./箱書きエディタ_0.11.0_amd64.AppImage
```
