# Release Notes - v0.20.1

## 🎉 新機能

### ✍️ 縦書き機能（実験的）
- **縦書きエディタの実装**: 執筆画面で縦書きによるプレビュー・編集が可能になりました。
- **TipTapによるモダンな編集体験**: 安定した縦書き入力を目指し、エディタ基盤をTipTapへ移行しました。

### 💾 保存プロセスの自動化
- **アプリ終了時の自動保存**: アプリを閉じる際に、自動的にプロジェクトの保存とテキストの書き出しを実行します。
- **「保存してから終了」の確認**: 予期せぬデータ損失を防ぐため、終了時の確認ダイアログを最適化しました。

### 📊 進捗管理の強化
- **総文字数カウント**: プロジェクト全体の合計文字数をリアルタイムで表示します。
- **当日進捗表示**: その日に執筆した文字数（増分）を確認できるようになりました。

### 📁 フォルダ構成の自動整理
- シーンの入れ替えや章タイトルの変更に合わせて、書き出し先のフォルダ名やファイル番号が自動的にリネーム・移動されるようになりました。
