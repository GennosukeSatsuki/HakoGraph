# HakoGraph - v1.0.0

祝！正式リリース！
「HakoGraph」が正式版 v1.0.0 に到達しました。

## [1.0.0] - 2025-12-20

### 追加・改善
- **モバイル版のドラッグ＆ドロップ操作の劇的改善**
  - シーンカードに専用のドラッグハンドル（⋮⋮）を追加しました。
  - ハンドル以外を操作することで、誤作動なくスムーズにリストをスクロールできるようになりました。
  - 長押し（200ms）によるドラッグ開始判定を最適化し、直感的な操作感を実現しました。
- **Android版の動作安定化**
  - Androidデバイス（Pixel 9等）でのエミュレーションと実機動作における互換性を向上させました。
  - タッチ操作時のレスポンスを向上させ、モバイル環境での執筆・構成体験を強化しました。
- **UI/UXのブラッシュアップ**
  - ドラッグ時の視覚フィードバックを改善し、カードの移動先がより分かりやすくなりました。

### 技術的な変更
- `@dnd-kit` のセンサー設定をモバイル/デスクトップで最適化（PointerSensorからMouse/TouchSensorへの移行）。
- `touch-action: none` をハンドルに適用し、ブラウザのデフォルトスクロールとの競合を解消。

---
これからも、より良い創作活動を支えるツールとして進化を続けていきます。
